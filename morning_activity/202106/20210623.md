# 作業概要

## 目標

[Todoist](https://todoist.com/app)のクローンアプリを作ることで以下を学ぶ

1. アプリを作る上で必要な React の知見
2. React のアプリ作成時のディレクトリ構成など
3. React Hooks や Redux の使い方
4. GraphQL の使い方
5. MongoDB の立て方
